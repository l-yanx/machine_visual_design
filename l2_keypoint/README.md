# l2_keypoint：角点检测与相似性匹配实验

## 1. 内容简介

- 在 **同一条处理管线** 下对比多种 **角点/关键点检测** 与 **相似性匹配** 方案：由输入图 A 经几何与 photometric 变换得到 B，再对 A、B **独立检测**，做描述子或邻域统计匹配，**左右拼接并连线**可视化。
- 各算法子工程均提供两种匹配方式，便于对比：
  - **ratio + mutual**（或邻域直方图 + 该策略，见 Harris）
  - **ratio + RANSAC**（单应性下几何一致性内点）
- 测试时统一参考各子目录 `README` 中 `**building.png`** 的实验数据（与算法目录下记录一致）。

## 3. 角点检测算法：原理与实验情况

以下 **实验数据** 均来自各算法目录 `README` 中 **测试图 `picture/building.png`**、**默认变换与默认参数** 下的一次记录（A/B 点数与匹配数以该表为准）。

### 3.1 Harris

- **原理要点**：在局部窗口用图像梯度构成二阶结构信息（Harris 响应或等价于 `goodFeaturesToTrack` 启用 Harris），得分高的点视为“角点”；可配合亚像素细化。本工程中匹配阶段使用 **角点邻域内的强度直方图 + 梯度方向直方图** 作描述，而非 Harris 本身自带的向量描述子。
- **数据（`building.png`）** | 角点 A=812，B=668。ratio+mutual 最终连线 19；ratio+RANSAC：初始匹配 31，内点/最终连线 19（见 `Harris/README` 第 5 节）。
- **情况简述**：两路最终 **连线数相同**；RANSAC 能显式给出 **31 → 19** 的筛减；主观上直方图匹配易出现 **误连、交叉线**，RANSAC 后视觉 **更合理**（见 `Harris/README` 第 6 节）。

### 3.2 ORB

- **原理要点**：**FAST** 在尺度金字塔上提点，**oBRIEF** 方向归一化后得到 **二进制描述子**；特点为速度快、可实时，匹配常用 **汉明距离**。
- **数据（`building.png`）** | 关键点 A=1200，B=1200。ratio+mutual 最终连线 538；ratio+RANSAC：初始 657，内点/最终 631（见 `ORB/README` 第 5 节）。
- **情况简述**：RANSAC 后 **有效连线数更高**；主观上 mutual 路仍有 **错误匹配与交叉线**，RANSAC 路 **几无明显错误**（见 `ORB/README` 第 6 节）。

### 3.3 SIFT

- **原理要点**：在 **多尺度高斯差分（DoG）** 空间找极值点，赋予 **主方向**，再于邻域内统计 **梯度方向直方图** 得到 128 维 **浮点描述子**；对尺度、旋转、光照变化通常更稳健。
- **数据（`building.png`）** | 关键点 A=1200，B=1201。ratio+mutual 最终连线 189；ratio+RANSAC：初始 229，内点/最终 200（见 `SIFT/README` 第 5 节）。
- **情况简述**：RANSAC 后连线数与 **内点数一致增加（189→200）** 的表述以 **RANSAC 最终连线 200、mutual 189** 为准；几何约束可 **剔除外点**（229→200）。

## 4. 相似性度量原理及实际效果

### 4.1 原理


| 环节            | 含义                                                                                                     |
| ------------- | ------------------------------------------------------------------------------------------------------ |
| **Harris 管线** | 用 **归一化邻域直方图**（强度 + 梯度方向）作向量；距离用与单位向量等价的 **欧氏/余弦** 关系；**Lowe ratio** 过滤“最近/次近”模糊匹配；**mutual** 要求互为最近邻。 |
| **ORB 管线**    | 使用 **ORB 二进制描述子**；`BFMatcher` + **汉明距离**；同样 **ratio + mutual**。                                        |
| **SIFT 管线**   | 使用 **SIFT 浮点描述子**；`BFMatcher` + **L2**；**ratio + mutual**。                                             |
| **RANSAC 分支** | 在 ratio 初匹配上，用 `**findHomography(..., RANSAC)`** 估计单应，按 **重投影误差阈值** 保留 **内点** 再连线，引入 **全局几何一致性**。      |


**Lowe ratio**：仅当“最近距离”明显小于“次近距离”时接受匹配，降低一对多/重复纹理歧义。  
**Mutual (cross-check)**：A→B 与 B→A 最近邻一致才保留，减少单边误配。  
**RANSAC + 单应性**：要求对应点整体满足同一平面（场景近似平面或视角近似）下的投影关系，**剔除**与模型不符的离群匹配。

### 4.2 实际效果（结合实例`building.png` ）

- **仅相似度（ratio+mutual 或直方图+mutual）**：连线数在 ORB 上 **最多**（538），但 **错配、交叉** 在 Harris/ORB 总结中均有描述。
- **加入 RANSAC**：Harris/ORB/SIFT 均呈现 **以几何为准的筛选**（Harris 31→19，ORB 657→631，SIFT 229→200），可视化上 **RANSAC 路通常更干净**。
- **横向粗比（同一张图、同套变换）**：ORB 的 **初匹配/内点数量** 最大；Harris+直方图 **最终可用连线** 在该次实验中最少；SIFT 介于其间，**描述子质量** 通常更利于论文级对比。

更细的命令、参数与表格见：`Harris/README.md`、`ORB/README.md`、`SIFT/README.md`。
